# Search & Filter Engine (FR4.0)

## Functional Description
The system shall allow users to search and filter products based on specific criteria.

Filtering options may include:
- Product Name
- Category
- Price Range
- Location
- Availability

## Actors
- Buyer / Consumer
- Vendor
- Farmer / Seller

## Preconditions
- Product listings exist in the database.

## Postconditions
- Filtered results are displayed accurately.

## Use Case Scenario

| Step | Actor | Action | System Response |
|------|-------|--------|----------------|
| 1 | User | Enters search keyword | System matches keyword |
| 2 | User | Applies filters | System narrows results |
| 3 | User | Views results | System displays filtered product list |

## Exceptions
- If no matching results are found, system displays: "No matching products found."
